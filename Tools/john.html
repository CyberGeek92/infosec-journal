Overview

John the Ripper (JtR) is a fast, flexible password cracker used to audit password strength and recover plaintext credentials from hashed values. It supports a wide range of hash formats and integrates with custom wordlists and cracking rules.

Core Capabilities

  • Crack password hashes using dictionary, brute-force, and hybrid attacks
  • Supports dozens of hash formats (MD5, SHA, NTLM, bcrypt, etc.)
  • Customizable cracking rules and masks
  • Supports incremental and single mode attacks
  • Works with hash dumps from /etc/shadow, SAM files, and CTF challenges

Basic Usage

john --wordlist=rockyou.txt hash.txt

Cracks hashes in hash.txt using the rockyou.txt wordlist.

Examples

Crack MD5 hashes with rockyou

john --wordlist=rockyou.txt --format=raw-md5 hash.txt

Crack NTLM hashes

john --wordlist=rockyou.txt --format=nt hash.txt

Show cracked passwords

john --show hash.txt

Benchmark performance

john --test

Brute-force with incremental mode

john --incremental hash.txt

Use a custom mask (e.g., 8-digit numeric)

john --mask='?d?d?d?d?d?d?d?d' hash.txt

Apply rules to mutate wordlist

john --wordlist=rockyou.txt --rules hash.txt

Pro Tips

  • Use john --list=formats to see all supported hash types
  • Use --rules to catch variations like Password123 or p@ssw0rd
  • Use --mask for targeted brute-force (e.g., PINs, phone numbers)
  • Use --format=auto to let John guess the hash type (not always reliable)
  • Use --fork=N to parallelize cracking across CPU cores

Output & Results

  • Cracked passwords are stored in ~/.john/john.pot
  • Use --show to display cracked results
  • Use --session=name to save and resume cracking sessions
