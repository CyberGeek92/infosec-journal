Overview

nmap (Network Mapper) is a powerful open-source tool used for network discovery, host enumeration, and vulnerability scanning. It’s a staple in penetration testing and red team operations.

Core Capabilities

  • Host discovery
  • Port scanning (TCP/UDP)
  • Service version detection
  • Operating system fingerprinting
  • Scriptable vulnerability scanning (via NSE)

Basic Scan

nmap <target>

Performs a default scan: top 1000 TCP ports with SYN scan.

Commonly Used Flags

  • -sS — TCP SYN scan (stealthy and fast)
  • -sT — TCP connect scan (less stealthy)
  • -sU — UDP scan
  • -p- — Scan all 65535 ports
  • -sV — Detect service versions
  • -O — OS detection
  • -A — Aggressive scan (OS + version + script + traceroute)
  • -Pn — Skip host discovery (treat host as online)
  • -T4 — Set timing template (T0–T5, higher = faster)
  • -v — Verbose output
  • -oN — Output to normal text file
  • -oX — Output to XML

Examples

Scan all ports and detect services

nmap -sS -sV -p- <target>

Aggressive scan with OS detection

nmap -A <target>

Scan multiple targets

nmap <target1> <target2> <target3>

Scan a subnet

nmap 192.168.1.0/24

Scan from a list of IPs

nmap -iL targets.txt

Save output to file

nmap -sV <target> -oN scan.txt

Advanced: Nmap Scripting Engine (NSE)

NSE allows you to run scripts for vulnerability detection, brute forcing, and more.

Run default scripts

nmap -sC <target>

Run specific script

nmap --script=http-title <target>

Run script category

nmap --script=vuln <target>

Pro Tips

  • Use -Pn if ICMP is blocked — it skips ping checks.
  • Use -T4 for faster scans, but avoid -T5 unless you know the target can handle it.
  • Use --open to show only open ports.
  • Use --reason to see why a port is marked open/closed.

Output Formats

  • -oN — Normal text
  • -oX — XML
  • -oG — Grepable
  • -oA — All formats at once

Useful NSE Script Categories

  • auth — Authentication bypass checks
  • default — Safe and useful default scripts
  • discovery — Network discovery scripts
  • vuln — Vulnerability detection
  • brute — Brute-force login attempts

Final Thoughts

nmap is more than just a port scanner — it’s a reconnaissance powerhouse. Mastering its flags and scripting engine gives you deep insight into any target. Whether you're mapping a network or probing for weaknesses, nmap is your first weapon.
